# CRM Frontend + Desktop Shell

React (Vite) UI and Electron packaging for **CRM Desktop**.

**Main documentation:** [../README.md](../README.md)

## Quick commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Browser dev server (port 5173) |
| `npm run electron` | Desktop dev (Vite + Electron + auto backend) |
| `npm run dist` | Production Windows installer |
| `npm run build:backend` | Build Spring Boot JAR into `resources/backend/` |
| `npm run download:jre` | Download bundled Temurin JRE |
